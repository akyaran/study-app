# 問題管理アプリ

プリントや問題集から作った問題リストを、プリント・ページ・大問単位でチェックするためのアプリです。

## 使い方

1. `manage-app/index.html` を開きます。
2. GitHub Pages上では、親フォルダの `questions.json` を自動で読み込みます。
3. 問題ごとに `できた`、`間違い`、`今日やる` を付けます。
4. 間違いが多い問題や `今日やる` にした問題が、左側の `今日やる候補` に出ます。
5. `エクスポート` と `復元` でブラウザ内の学習管理データを移せます。

## 読み込みJSON形式

読み取りAIなどで別途作る場合は、次のようなJSONを読み込めます。

```json
[
  {
    "id": "math-division-page15-q1",
    "categoryId": "math-division",
    "categoryLabel": "算数（割り算）",
    "source": "2026-05-22 18-36 ページ 15.JPEG",
    "itemNumber": "たしかめよう 1",
    "title": "次の計算をしましょう。また、たしかめも書きましょう。",
    "note": "914 ÷ 5、888 ÷ 9、864 ÷ 8"
  }
]
```

既存の `questions.json` 形式も読み込めます。
