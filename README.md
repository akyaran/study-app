# 勉強アプリ

iPadでも使いやすい、シンプルな自習用アプリです。問題を見て紙などに解答し、アプリ内で解答と解説を確認します。

## 使い方

1. `index.html` をブラウザで開きます。
2. 右上の「読み込み」から `questions.json` を選びます。
3. 「ヒント」「解答へ」「解説」を使って学習します。

GitHub Pagesなどで公開する場合は、`index.html`, `styles.css`, `app.js`, `questions.json`, `images` を同じ場所に置けば、`questions.json` が自動で読み込まれます。

## 問題データ

問題データの作り方は `DATA_FORMAT.md` を見てください。問題、ヒント、解答、解説はテキストでも画像でも表示できます。
